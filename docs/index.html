LinkedIn Outreach

AI-powered outreach messages for LinkedIn profiles

LinkedIn Outreach is a Chrome extension that analyzes LinkedIn profiles and generates personalized outreach messages using OpenAI's GPT-4o-mini model.

Features

  • 🔍 Extracts name, bio, location, about section, and recent posts from any LinkedIn profile
  • 🤖 Generates friendly, professional, personalized messages via the OpenAI API
  • 📋 One-click paste into LinkedIn's DM compose box
  • 🔑 Stores your OpenAI API key locally – never sent to our servers
